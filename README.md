# Conplext

This repository contains Conplext, a multilingual dataset designed for lexical complexity prediction (LCP) in the context of foreign/second language (L2) learning. The resource covers 3,901 sentence contexts for 1,000 vocabulary items across five languages (English, French, Spanish, Swedish, and Dutch), each aligned with Common European Framework of Reference (CEFR) proficiency levels. Contexts were generated using a generative large language model (LLM). At the time of writing, a large-scale best–worst scaling (BWS) annotation experiment is being conducted with L2 learners to derive continuous, learner-informed lexical complexity values for the 3,901 sentence contexts included in the corpus. The resulting dataset enables the development of context-aware word difficulty models that account for variation across both languages and learning stages. In addition to its primary use in LCP, Conplext provides valuable opportunities for research in word sense disambiguation, generative model evaluation, and adaptive language learning applications.

Below you can find a brief description of the dataset files included in the present repository. For more details on the dataset, please refer to the corresponding paper presented at the DeTermIt! workshop co-located with LREC2026 (Alfter & Degraeuwe, in press). Please note that the dataset can be used for research purposes only.

- `randomSamples`: This directory contains the five files presenting the 1,000 vocabulary items selected for Conplext. Each file represents a separate language and includes the 200 vocabulary items that were, according to a random stratified sampling method, selected for that language from the corresponding [CEFRLex](https://cental.uclouvain.be/cefrlex/) resource. Per applicable CEFR level (A1-A2-B1-B2-C1), 40 items were selected (20 nouns and 20 verbs).
- `generatedSentences`: This directory contains the files including the generated sentence contexts. For each of the 200 vocabulary items per target language, five sentence contexts were generated (one per CEFR level). The files ending in `_all.tsv` contain, per language, the original sentences as generated by the large language model. The files ending in `_filtered.tsv` contain the filtered versions ("Too Difficult" instances removed and duplicate instances collapsed into a single entry).
- `BWS_4-tuples`: This directory contains the 4-tuples used in the ongoing BWS experiment (separate file per part-of-speech category).
